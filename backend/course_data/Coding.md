# Coding for Kids - Vikalp Online School

## Overview
Explore coding at Vikalp India, where young learners embark on a fun and educational journey into the world of programming. Empower your child with valuable coding skills for the future.

## Course Levels
### Beginners Level
- Age: 6-8 years
- Introduction to computational thinking
- Block-based programming (Scratch)
- Simple game creation
- Duration: 3 months

### Intermediate Level
- Age: 9-11 years
- Advanced Scratch projects
- Introduction to Python basics
- Interactive stories and animations
- Duration: 3 months

### Advanced Level
- Age: 12-14 years
- Python programming
- Web development basics (HTML, CSS)
- Game development
- Duration: 3 months

### Professional Level
- Age: 14+ years
- Advanced Python/JavaScript
- App development
- Database basics
- Real-world project development
- Duration: 3 months

## Course Structure
- **Duration**: 3 months per level (Full course: 12 months)
- **Classes per week**: 2 sessions
- **Platform**: Zoom with screen sharing
- **Batch size**: 6-8 students

## Fee Structure
- **Per Level**: ₹15,000 - ₹20,000
- **Full Course (4 levels)**: ₹50,000 - ₹70,000 (discount applied)
- **Demo Class**: Free trial available

## FAQs
**Q: What is the minimum age?**
A: Children from age 6 can start with Beginners Level.

**Q: Do students need a laptop?**
A: Yes, a laptop/desktop with internet connection is required.

**Q: What programming languages are taught?**
A: Scratch (beginners), Python, HTML/CSS, JavaScript (advanced).

**Q: Are there projects?**
A: Yes, each level culminates with a project that students build independently.

## Learning Outcomes
- Logical thinking and problem-solving
- Computational thinking skills
- Ability to create games and apps
- Foundation for future tech careers
